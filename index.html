About Jon

I am a software developer, focused on .NET and MERN, full-stack development.

My passion is to find simple solutions to high friction problems such as user interface, or collection, copiling, and displaying data, on Mobile, Web, and Desktop platforms.

Having taken a unique path to software development, I bring experiences of a systems administrator, a computer network technician, and a software test engineer into my designs and solutions.

My favorite tools are C# and DotNET including ASP.NET Blazor and .NET MAUI. I enjoy building web-sites using frameworks like React, and web servers using Express.js or Next.js.

On a regular basis, I challenge myself to learn new techniques and language details to constantly improve vs my yesterday.
